# C4 GYM - Premium Fitness Website 🏋️‍♂️

A premium, mobile-first responsive website for **C4 GYM** located in Rawalpindi, Pakistan.

## 🔥 Live Features

- **Mobile-First Design** - Optimized primarily for mobile users with full PC support
- **Premium Dark Theme** - Black & red gym-inspired color scheme
- **Motivational Quotes** - Painful and inspiring quotes throughout the site
- **Interactive Membership Plans** - 1 Month, 3 Months, and 6 Months plans with popup
- - **Smooth Animations** - Scroll-triggered animations, particle effects, counter animations
- **Social Media Integration** - Instagram and Facebook follow buttons
- **One-Click Actions** - Call, Text, Directions, and Share functionality
- **Responsive Navigation** - Full-screen mobile menu with smooth transitions
